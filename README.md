# Gym Management System

A web-based Gym Management System developed using PHP and MySQL.

## Features
- Admin Login
- Dashboard
- Add Members
- Manage Members
- Payments Management
- Attendance Tracking
- Reports

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Setup Instructions

1. Install XAMPP
2. Copy project folder to htdocs
3. Import database.sql in phpMyAdmin
4. Open browser and run:

http://localhost/gym-management/login.php
